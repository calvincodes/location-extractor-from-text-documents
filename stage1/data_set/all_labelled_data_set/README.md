# Entire DataSet for Training and Testing

## Entity Type
* Location

## Tag Description
* Locations are marked up in the documents using **\<loc\>....\</loc\>** tags.

## Tag Examples

### Positives
1. \<loc\>India\</loc\>
2. \<loc\>United States\</loc\>

### Negatives

2. The _United States_ attorney
